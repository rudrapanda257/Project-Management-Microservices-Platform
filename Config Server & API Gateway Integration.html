2

3

JwtAuthenticationFilter validates the token. Extracts user info (email, role, userId)
4

Gateway queries Eureka: "Where is PROJECT-SERVICE?" ‚Üí Gets "localhost:8082"

7

Gateway forwards response back to client with 201 Created status

üéØ Key Benefits of Day 6 Architecture
1. Centralized Configuration: Change database URL in one place, all services pick it up
2. Single Entry Point: Clients only need to know port 8080
3. JWT at Gateway: Authentication happens once at gateway, not in every service
4. Load Balancing: Gateway can route to multiple instances of same service
5. Service Discovery: Services can move, gateway finds them automatically
6. Configuration Versioning: All configs in Git with full history
